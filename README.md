# Digital-Insurance-Management-System
The Digital Insurance Management System is a desktop application developed to manage insurance policies, clients, and records in an organized and efficient way. The system provides a user-friendly interface that allows insurance staff to register clients, store important details, and quickly retrieve information when needed.

Built using JavaFX for the graphical user interface and MySQL for database management, the system ensures reliable data storage and smooth interaction between the application and the database through JDBC connectivity.

Key Features
Add new client insurance records
Search clients using their RSA ID number
Validate RSA ID numbers (must contain exactly 13 digits)
Update existing insurance details
Delete client records
View stored insurance data in a structured format
Secure database connection using MySQL
Technologies Used
Java (Core Logic)
JavaFX (User Interface)
MySQL (Database)
JDBC (Database Connectivity)
Purpose

This project was developed to demonstrate practical skills in database integration, GUI development, and data validation within a real-world insurance management scenario. It focuses on improving efficiency, reducing manual record handling, and ensuring accurate data management.
